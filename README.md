# 目标
- 结构化收支记录
- 使用 CI 自动化得到余额和资金收支报告

# 账目约定
- 收付款方
    - 委员会，指“科协未来星计划”管理委员会
    - 协会，指科技创新协会
- 用途
    - 资助，指资方入资，此时金额必须为正值
    - 执行，指协会使用资金，此时金额必须为负值
